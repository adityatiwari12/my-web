ASTITAVA & NyayaSetu

AI-Driven Forest Governance and Decision Support Platform

Project Overview

ASTITAVA is an AI-powered WebGIS platform designed to digitize, analyze, and visualize forest land governance data for tribal communities and government authorities. The system addresses thousands of paper-based Forest Rights Act (FRA) documents that are slow to process and difficult to analyze at scale.

Key Achievements

  • Digitized 1,000+ documents with 85% reduction in processing time
  • OCR + NER pipeline for automated entity extraction
  • WebGIS mapping with spatial analytics
  • Decision support dashboards for policymakers

Technical Stack

Frontend: React, TypeScript, WebGIS
Backend: FastAPI, PostgreSQL
AI: OCR, NER models

Stellar 3D

Interactive Solar System Explorer with AI-Driven Cinematic Tours

Project Overview

STELLAR 3D is an interactive, real-time 3D solar system explorer combining physics-based orbital simulations, cinematic AI-generated tours, and a conversational space-knowledge chatbot powered by NASA datasets.

Core Features

  • Real-Time 3D Simulation: Physically accurate orbital mechanics with adjustable time scale
  • AI Cinematic Tours: Guided experiences of eclipses, alignments, and astronomical events
  • Eclipse Simulator: Solar/lunar eclipse visualization with shadow cones
  • AI Chatbot: RAG-based assistant trained on NASA datasets

Technical Stack

Frontend: Three.js, React
Physics: Orbital mechanics engine
AI: RAG chatbot with NASA data
Backend: Space data retrieval API

Impact

Makes complex astronomical phenomena intuitive for students, educators, and space enthusiasts through immersive visualization and AI-powered learning.
